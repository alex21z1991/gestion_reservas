# ClickyGO - Sistema de Gestión de Reservas para Restaurantes

## Descripción General

ClickyGO es una aplicación web diseñada para optimizar la gestión de reservas en pequeños y medianos restaurantes. La plataforma permite administrar horarios, disponibilidad de mesas y reservas de clientes de manera eficiente, reduciendo errores asociados a procesos manuales y mejorando la experiencia tanto para los clientes como para el personal del negocio.

El sistema fue desarrollado aplicando metodologías ágiles basadas en Scrum, utilizando Historias de Usuario, planificación de Sprint y estimación mediante Planning Poker para garantizar un desarrollo organizado y orientado a las necesidades reales de los usuarios.

## Problema que Resuelve

Muchos restaurantes gestionan sus reservas mediante llamadas telefónicas, mensajes o registros manuales, lo que puede generar:

* Duplicidad de reservas.
* Errores en la asignación de horarios.
* Pérdida de información de clientes.
* Dificultad para administrar la disponibilidad de mesas.
* Retrasos en la atención y organización del negocio.

ClickyGO centraliza toda esta información en una única plataforma, permitiendo una administración más eficiente y segura.

## Objetivos del Proyecto

El objetivo principal es proporcionar una herramienta digital que facilite la administración de reservas mediante:

* Registro y autenticación de usuarios.
* Consulta de horarios disponibles.
* Creación y cancelación de reservas.
* Gestión administrativa de disponibilidad.
* Administración de negocios y servicios.
* Prevención de conflictos por reservas duplicadas.

## Roles del Sistema

### Cliente

Puede:

* Registrarse en la plataforma.
* Iniciar sesión.
* Consultar horarios disponibles.
* Crear reservas.
* Cancelar reservas existentes.

### Administrador

Puede:

* Gestionar horarios disponibles.
* Supervisar reservas realizadas.
* Administrar la disponibilidad general del sistema.
* Aprobar y validar negocios registrados.

### Dueño del Negocio

Puede:

* Registrar su restaurante.
* Configurar servicios ofrecidos.
* Gestionar horarios y disponibilidad.
* Supervisar las reservas asociadas a su negocio.
